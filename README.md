# Temperature Converter

A responsive **Temperature Converter** web application built using **HTML, CSS, and JavaScript**. This project allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin instantly.

##  Features

- Convert temperatures between Celsius, Fahrenheit, and Kelvin
- Real-time conversion on input
- Responsive and user-friendly design
- Input validation to ensure correct numbers
- Instant display of conversion results

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

##  How It Works

1. Users enter a temperature value and select the input unit (Celsius, Fahrenheit, Kelvin).
2. JavaScript calculates the equivalent temperatures in the other units.
3. The results are displayed instantly on the page.
4. Input validation ensures only valid numeric values are accepted.

##  How to Use

1. Clone the repository:
   ```
   git clone https://github.com/Fida-ur-Rahman/Temperature-Converter.git
   ```
2. Open the project folder.
3. Run `index.html` in your browser.
4. Enter a temperature, select the unit, and see the conversion instantly.
